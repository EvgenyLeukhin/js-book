# Install

## Install 1 (create-next-app)

1. ```npx create-next-app nextjs-blog --use-npm --example "https://github.com/vercel/next-learn/tree/master/basics/learn-starter"```

2. ```cd nextjs-blog```

3. ```npm run dev```


## Install 2 (manual)

1. ```npm init -y```

2. ```npm i --save react react-dom next```

3. ./pages/index.js``

4. package.json

```json
  "scripts": {
    "start": "next dev",
    "build": "next build",
    "lint": "next lint"
  },
```
