# Number

## Exact & fractional

```js
let exact = 12;

let fractional1 = 12.25;

let fractional = 12,25; // Error

let fractional = .25; // equal to 0.25

```

Infinity и NaN - это не отдельные типы, а подразновидность number.

## Infinity

```js
123 / 0;
```

## NaN

```js
123 / 'String';

'String' / 2;
```
