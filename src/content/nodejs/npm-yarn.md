# Npm / Yarn

Npm install with node automaticly.

Install:

```bash
npm i -g package-name
npm i --save-dev package-name
```

Remove:

```bash
npm remove -g package-name
npm remove --save-dev package-name
```

## Yarn

Install:

```bash
npm i -g yarn
yarn -v
```

```bash
yarn add package-name
```

Remove:

```bash
yarn remove package-name
```
